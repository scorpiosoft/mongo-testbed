# mongo-testbed
A sandbox in which to test all things MongoDB
Uses React for display

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com)
- [yarn](https://www.npmjs.com/package/yarn)

### Installing

1. Clone this repo down to your local machine
2. `yarn install`

### Running

1. `yarn start`
